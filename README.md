# Controle-estoque-automatizado-Excel
Gestão de estoque automatizada em Excel com interface de software (UX), alertas de compras e dashboard dinâmico.

Este sistema de gestão dispõe de: 

1 - Planilha "Início": 
Nesta planilha, estão as orientações de utilização. Com interface de software, as formas "Início", "Cadastro" e "Lançamentos" possuem hiperlinks que levam de uma aba a outra, facilitando a navegação para o usuário.

2 - Planilha "Cadastro": 
Nesta planilha, os dados estão em tabela, facilitando a utilização e manipulação. Essas tabelas possuem filtros e linha de totais. Além disso, dispõe também das fórmulas SOMASE e SOMASES para atualização dinâmica do saldo, a fórmula SE e formatação condicional para gerar avisos de "Solicitar nova compra" quando o saldo estoque estiver abaixo do mínimo e "Priorizar venda!" quando estiver a cima do estoque máximo, e por fim, para otimização das informações, há também um Dashboard indicando a composição atual do estoque. 

3 - Planilha "Lançamentos": 
Nesta planilha, para realização de lançamentos, o sistema utiliza validação de dados através de uma lista suspensa, focando apenas em produtos realmente cadastrados, evitando erros de lançamentos. Além disso, também possuí as fórmulas SOMASE, SOMASES e formatação condicional para situações específicas, como por exemplo, saldo negativo.
